# Benjamin Laffitte Academic Homepage

Static academic homepage for Benjamin Laffitte, Assistant Professor and Earth System and Carbon-Cycle Scientist at Chengdu University of Technology.

## Before publishing

The homepage uses `portrait.jpg` as the professional photograph, local images in `assets/about/` for the About photo gallery, `assets/news/eu-alumni-chengdu.jpg` for the EEAS Alumni Homecoming news item, and local research-figure images for the Nature Communications, Agricultural and Forest Meteorology, and Global Change Biology news items. The canonical, structured-data, sitemap, and robots URLs are set for `https://benjaminlaffitte.github.io/`.

## Publish using GitHub Pages

1. Create a GitHub account.
2. Create a public repository named exactly `benjaminlaffitte.github.io`.
3. Upload all files from this folder to the repository root.
4. Open Settings > Pages.
5. Under Build and deployment, choose Deploy from a branch.
6. Select the `main` branch and `/ (root)`, then save.
7. Visit `https://benjaminlaffitte.github.io/`.

## Google Search Console

After the page is online:

1. Add the site as a Search Console property.
2. Verify ownership.
3. Inspect the homepage URL.
4. Select Request indexing.
5. Submit `sitemap.xml` through the Sitemaps section.

## Custom domain

You can later connect a domain such as `benjaminlaffitte.com` from Settings > Pages > Custom domain. Update every canonical, Open Graph, sitemap, robots.txt, and structured-data URL to the final domain.
